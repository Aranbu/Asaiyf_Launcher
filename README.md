# Asaiyf_Launcher
Are you tired of having a cluttered desktop? With Asaiyf (Pronounced As-Sayf) Launcher, you can have all of your shortcuts organized and run them with a lightweight minimalist launcher, no need to clutter your desktop!

* **Step 1**: Install Asaiyf Launcher
* **Step 2**: In the folder where you installed the launcher, open "Asaiyf Shortcuts" and drag in there any files (preferably shortcuts of said files rather than the original files) that you want to launch
* **Step** 3: Run Asaiyf Launcher, select what you want to launch in the dropdown, and hit "Run"!

*If you change the contents of the folder while the launcher is still running, click "Update List" to refresh the options in the dropdown.*
